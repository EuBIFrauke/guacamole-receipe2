
# A recipe for Guacamole

Make sure to enjoy!

- avocado
- onion
- salt & pepper
- tomatoes

# Instructions

- chop onions and tomatoes
- cut open avocado and remove stone
- scoop out flesh
- put into a bowl and mix with tomatoes and onion
